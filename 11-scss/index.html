Les bases de SASS et SCSS

  • Installer SASS va npm : "npm install -g sass"
  • Pour vérifier quil est bien installé, lancer "sass" ou "npw sass"
  • Si l'éxecution des scripts est bloqué, lancer dans powershell en administrateur "set-executionpolicy unrestricted"
  • Les différentes commandes :
    • sass pathToFile.scss pathToFile.css
    • sass --watch pathToFile.scss pathToFile.css
    • sass --watch sassFolder:cssFolder
    • sass --watch sassFolder:cssFolder --stylecompressed
